# docker container and system monitoring

a docker-compose quickstart for docker container and system monitoring

## tech stack

* docker/docker-compose
* prometheus
* node-exporter
* cadvisor
* grafana

## setup

```
$ docker-compose up
```

## login

```
http://localhost:3000
admin/admin
```
